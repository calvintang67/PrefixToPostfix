Version: python 3.11.9
IDE: VSCode, python extension installed
Author: Calvin Tang
Purpose: Convert prefix expressions to postfix expressions

When inputting command line prompt, follow the following format:
python "main.py" "inputFile.txt" "outputFile.txt"

(1) "main.py" is the main script that runs my program
(2) "inputFile.txt" is a file name specified by the user. Write lines of prefix code in text file to process expression in program and write on "outputFile.txt:
(3) "outputFile.txt" is a file specified by the user. All converted postfix expressions will be written onto this text file.
